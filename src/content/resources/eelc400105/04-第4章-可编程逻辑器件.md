# 4.2 简单可编程逻辑SPLD

分类：
1. ROM（Read  Only  Memory）
2. PLA（Programmable  Logic Array ）
3. PAL（Programmable  Array  Logic ）
4. GAL（Generic  Logic Array ）

## 4.2.1 ROM

一个只能在制造（或编程）时方可将信息写入，而在应用时仅能读出的存储器。（英特尔公司发明）

与阵列：固定
或阵列：可编程
![第4章 可编程逻辑器件__3](https://pub-e7c93c0b727d457c895b016a26707158.r2.dev/2026/07/%E7%AC%AC4%E7%AB%A0%20%E5%8F%AF%E7%BC%96%E7%A8%8B%E9%80%BB%E8%BE%91%E5%99%A8%E4%BB%B6__3.png)

## 4.2.2 PLA

与、或阵列都可以编程，方便实现逻辑函数的最简与或式。
![第4章 可编程逻辑器件__4](https://pub-e7c93c0b727d457c895b016a26707158.r2.dev/2026/07/%E7%AC%AC4%E7%AB%A0%20%E5%8F%AF%E7%BC%96%E7%A8%8B%E9%80%BB%E8%BE%91%E5%99%A8%E4%BB%B6__4.png)

## 4.2.3 PAL

1. 与阵列可编程，或阵列是固定的
2. 分为组合PAL（输出不带寄存器）和时序PAL

早期PAL是OTP，现代有Flash，SRAM等工艺

![第4章 可编程逻辑器件__5](https://pub-e7c93c0b727d457c895b016a26707158.r2.dev/2026/07/%E7%AC%AC4%E7%AB%A0%20%E5%8F%AF%E7%BC%96%E7%A8%8B%E9%80%BB%E8%BE%91%E5%99%A8%E4%BB%B6__5.png)

如果想把输出额低有效变成高有效，只需要根据原来函数的反函数对与阵列进行编程即可。

对于PAL来说，如果与项的数目已经小于与阵列的数目，那就没有必要继续进行了。

![第4章 可编程逻辑器件__6](https://pub-e7c93c0b727d457c895b016a26707158.r2.dev/2026/07/%E7%AC%AC4%E7%AB%A0%20%E5%8F%AF%E7%BC%96%E7%A8%8B%E9%80%BB%E8%BE%91%E5%99%A8%E4%BB%B6__6.png)
