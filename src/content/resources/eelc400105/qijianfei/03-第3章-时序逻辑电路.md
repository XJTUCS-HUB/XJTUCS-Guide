# 3.1 时序逻辑电路
## 3.1.1 概述

**逻辑电路分类**：
1. 组合逻辑电路
2. 时序逻辑电路

逻辑电路是接收输入信号且产生与输入信号有确定关系的正确且稳定的输出信号。

时序电路的状态是状态变量的集合，它在任何时候的值都包含对确定电路将来行为的过去信息。

![第3章 时序逻辑电路__1](https://pub-e7c93c0b727d457c895b016a26707158.r2.dev/2026/05/%E7%AC%AC3%E7%AB%A0%20%E6%97%B6%E5%BA%8F%E9%80%BB%E8%BE%91%E7%94%B5%E8%B7%AF__1.png)

n个$y_i$变量可以定义$2^n$个状态，用二进制的形式表示出来，其中的每一位数称为一个状态变量。

### 时序电路的分类

#### 1）根据引起状态变化的原因分类

**同步时序电路**：
- 在统一时钟CP/CLK控制下同步改变状态
- 所有触发器在同一时钟脉冲作用下同时翻转
- 抗干扰能力强，设计规范

**异步时序电路**：
- 没有统一的时钟脉冲同步
- 状态转换由输入信号直接触发
- 速度较快但易受干扰

#### 2）按输入信号x的特性分类

![第3章 时序逻辑电路__2](https://pub-e7c93c0b727d457c895b016a26707158.r2.dev/2026/05/%E7%AC%AC3%E7%AB%A0%20%E6%97%B6%E5%BA%8F%E9%80%BB%E8%BE%91%E7%94%B5%E8%B7%AF__2.png)

#### 3）按输出信号的特性分类

**Mealy型时序电路**：
- 输出不仅与当前状态有关，还与当前输入有关
- 输出方程：$Z(t) = F[X(t), Y(t)]$
- 响应速度快

**Moore型时序电路**：
- 输出仅与当前状态有关
- 输出方程：$Z(t) = F[Y(t)]$
- 设计简单，稳定性高

![第3章 时序逻辑电路__3](https://pub-e7c93c0b727d457c895b016a26707158.r2.dev/2026/05/%E7%AC%AC3%E7%AB%A0%20%E6%97%B6%E5%BA%8F%E9%80%BB%E8%BE%91%E7%94%B5%E8%B7%AF__3.png)

# 3.2 时序电路的双稳态元件

双稳态元件的特点：
1. 有两个稳定状态，分别表示0或1
2. 在激励信号的作用下可以发生稳态的翻转

**现态**（Present State, $Q^n$）：触发器当前所处的状态
**次态**（Next State, $Q^{n+1}$）：触发器在激励信号作用后转换到的状态
有两个互补的输出端，分别称为1态和0态

**锁存器与触发器的区别**：

| 类型  | 触发方式    | 时钟控制 | 功能特点            |
| --- | ------- | ---- | --------------- |
| 锁存器 | 电平触发    | 无    | 由门电路直接构成，具有锁存功能 |
| 触发器 | 脉冲/边沿触发 | 有    | 由锁存器构成，具有保持功能   |

触发器包括：
- 主从结构的脉冲触发器
- 维持阻塞结构的边沿触发器

### 3.2.1 S-R锁存器

![第3章 时序逻辑电路__6](https://pub-e7c93c0b727d457c895b016a26707158.r2.dev/2026/05/%E7%AC%AC3%E7%AB%A0%20%E6%97%B6%E5%BA%8F%E9%80%BB%E8%BE%91%E7%94%B5%E8%B7%AF__6.png)

**S-R锁存器特性**：
- **S置位端**（Set）：S=1时，Q=1
- **R复位端**（Reset）：R=1时，Q=0
- **约束条件**：S和R不能同时为1，$S·R=0$

**真值表**：

| S   | R   | Qn  | Qn+1 | 功能 |
| --- | --- | --- | ---- | ---- |
| 0   | 0   | 0   | 0    | 保持 |
| 0   | 0   | 1   | 1    | 保持 |
| 0   | 1   | 0   | 0    | 复位 |
| 0   | 1   | 1   | 0    | 复位 |
| 1   | 0   | 0   | 1    | 置位 |
| 1   | 0   | 1   | 1    | 置位 |
| 1   | 1   | 0   | 不定 | 禁止 |
| 1   | 1   | 1   | 不定 | 禁止 |

**次态方程**：
$$Q^{n+1} = S + \overline{R}·Q^n \quad (\text{约束条件：} S·R=0)$$

具体的分析可以参考[ppt](https://lms.xjtu.edu.cn/course/27524/learning-activity/full-screen#/874963)

### 3.2.2 /S-/R锁存器

![第3章 时序逻辑电路__7](https://pub-e7c93c0b727d457c895b016a26707158.r2.dev/2026/05/%E7%AC%AC3%E7%AB%A0%20%E6%97%B6%E5%BA%8F%E9%80%BB%E8%BE%91%E7%94%B5%E8%B7%AF__7.png)

与S-R锁存器类似，区别在于有效电平变为低有效，或非门变为与非门。

**次态方程**：
$$Q^{n+1} = \overline{/S} + /R·Q^n$$

### 3.2.3 带使能端的S-R锁存器

![第3章 时序逻辑电路__9](https://pub-e7c93c0b727d457c895b016a26707158.r2.dev/2026/05/%E7%AC%AC3%E7%AB%A0%20%E6%97%B6%E5%BA%8F%E9%80%BB%E8%BE%91%E7%94%B5%E8%B7%AF__9.png)

在使能信号EN有效时，其功能与基本S-R锁存器相同；在EN无效时，电路保持原状态不变。

**次态方程**：
$$Q^{n+1} = EN·(S + \overline{R}·Q^n) + \overline{EN}·Q^n$$

### 3.2.4 D锁存器

![第3章 时序逻辑电路__10](https://pub-e7c93c0b727d457c895b016a26707158.r2.dev/2026/05/%E7%AC%AC3%E7%AB%A0%20%E6%97%B6%E5%BA%8F%E9%80%BB%E8%BE%91%E7%94%B5%E8%B7%AF__10.png)

S端和R端不会出现同时为1的情况，一般用来保存二进制数。

$S = D$，$R = \overline{D}$

**次态方程**：
$$Q^{n+1} = D + \overline{D}·Q^n = D$$

### 3.2.5 边沿触发的D触发器

![第3章 时序逻辑电路__11](https://pub-e7c93c0b727d457c895b016a26707158.r2.dev/2026/05/%E7%AC%AC3%E7%AB%A0%20%E6%97%B6%E5%BA%8F%E9%80%BB%E8%BE%91%E7%94%B5%E8%B7%AF__11.png)

**结构说明**：
- 门1、2实现/S-/R锁存器
- 门3、4实现使能作用
- 门5、6生成D和$\overline{D}$，保证3、4的输入端互补

**维持阻塞线**：
- **a线**：置0阻塞线
- **b线**：置1维持线
- **c线**：置0维持线

**/PR**（Preset）：异步置1端
**/CLR**（Clear）：异步置0端
- /PR和/CLR一般在电路系统初始化时使用

### 3.2.7 主从J-K触发器

![第3章 时序逻辑电路__16](https://pub-e7c93c0b727d457c895b016a26707158.r2.dev/2026/05/%E7%AC%AC3%E7%AB%A0%20%E6%97%B6%E5%BA%8F%E9%80%BB%E8%BE%91%E7%94%B5%E8%B7%AF__16.png)
![第3章 时序逻辑电路__17](https://pub-e7c93c0b727d457c895b016a26707158.r2.dev/2026/05/%E7%AC%AC3%E7%AB%A0%20%E6%97%B6%E5%BA%8F%E9%80%BB%E8%BE%91%E7%94%B5%E8%B7%AF__17.png)

**工作特点**：
- 解决了S-R触发器的约束条件问题
- 当J和K同时为1时，具有翻转功能，可用于二进制计数器

**次态方程**：
$$Q^{n+1} = J·\overline{Q} + \overline{K}·Q$$

**时序要求**：
- CLK高电平至少维持2个门的时间（带使能的S-R锁存器）
- 下降沿又要2个门的时间用于输出
- 触发脉冲每个时间间隔要大于4个门的时间

![第3章 时序逻辑电路__18](https://pub-e7c93c0b727d457c895b016a26707158.r2.dev/2026/05/%E7%AC%AC3%E7%AB%A0%20%E6%97%B6%E5%BA%8F%E9%80%BB%E8%BE%91%E7%94%B5%E8%B7%AF__18.png)

### 3.2.8 边沿触发J-K触发器

边沿触发J-K触发器类似于D触发器，也有建立时间和保持时间。JK触发器常用于同步时序电路中。

![第3章 时序逻辑电路__19](https://pub-e7c93c0b727d457c895b016a26707158.r2.dev/2026/05/%E7%AC%AC3%E7%AB%A0%20%E6%97%B6%E5%BA%8F%E9%80%BB%E8%BE%91%E7%94%B5%E8%B7%AF__19.png)

**次态方程**：

$$Q^{n+1} = D = J·\overline{Q} + \overline{K}·Q$$

**功能表**：

| J | K | Qn | Qn+1 | 功能 |
| --- | --- | --- | ---- | ---- |
| 0 | 0 | 0 | 0 | 保持 |
| 0 | 0 | 1 | 1 | 保持 |
| 0 | 1 | 0 | 0 | 复位 |
| 0 | 1 | 1 | 0 | 复位 |
| 1 | 0 | 0 | 1 | 置位 |
| 1 | 0 | 1 | 1 | 置位 |
| 1 | 1 | 0 | 1 | 翻转 |
| 1 | 1 | 1 | 0 | 翻转 |

### 3.2.9 T触发器

![第3章 时序逻辑电路__20](https://pub-e7c93c0b727d457c895b016a26707158.r2.dev/2026/05/%E7%AC%AC3%E7%AB%A0%20%E6%97%B6%E5%BA%8F%E9%80%BB%E8%BE%91%E7%94%B5%E8%B7%AF__20.png)

每来一个计数脉冲CLK，输出就变反一次。

T触发器可以通过将J-K触发器的J端和K端连接在一起实现。
![第3章 时序逻辑电路__21](https://pub-e7c93c0b727d457c895b016a26707158.r2.dev/2026/05/%E7%AC%AC3%E7%AB%A0%20%E6%97%B6%E5%BA%8F%E9%80%BB%E8%BE%91%E7%94%B5%E8%B7%AF__21.png)

**功能**：
- T=0时：保持原状态
- T=1时：状态翻转

**次态方程**：
$$Q^{n+1} = T·\overline{Q} + \overline{T}·Q = T \oplus Q$$

**功能表**：

| T | Qn | Qn+1 | 功能 |
| --- | --- | ---- | ---- |
| 0 | 0 | 0 | 保持 |
| 0 | 1 | 1 | 保持 |
| 1 | 0 | 1 | 翻转 |
| 1 | 1 | 0 | 翻转 |

T触发器常用于计数器和分频电路。

一些场合也使用无使能控制的T触发器。

### 3.2.10 触发器功能比较

| 类型     | 约束条件  | 特性方程                                  | 主要应用    |
| ------ | ----- | ------------------------------------- | ------- |
| S-R锁存器 | S·R=0 | $Q^{n+1}=S+\overline{R}·Q^n$          | 基本存储    |
| D锁存器   | 无     | $Q^{n+1}=D$                           | 数据暂存    |
| D触发器   | 无     | $Q^{n+1}=D$                           | 移位寄存器   |
| J-K触发器 | 无     | $Q^{n+1}=J\overline{Q}+\overline{K}Q$ | 计数器、控制器 |
| T触发器   | 无     | $Q^{n+1}=T\oplus Q$                   | 计数器、分频器 |

# 3.3 同步时序电路的分析与设计
## 3.3.1 同步时序电路的分析方法

分析工作从组合逻辑的分析着手：
1. 列出激励函数及输出函数表达式
2. 根据触发器的次态方程得到各个状态变量的次态方程
$$\text{次态}=Q(\text{输入},\text{现态})$$
3. 画出二进制状态表
4. 填写输出值，得到二进制状态/输出表
5. 在上一步基础上得到状态/输出表
6. 画出状态图
7. 电路特性描述，逻辑功能分析

建议结合ppt案例具体分析，这里不再赘述。

### 2 状态化简

#### 1）完全给定同步时序电路状态表的化简

**状态化简法**

状态等效定义：

在同一电路中，等效状态可以合并为一个状态。等效具有传递性，通过这个性质我们找出所有的最大等效类，从而实现状态化简。

等效对的判断标准：
1. 输出完全相同
2. 次态满足：相同/交错/维持/循环，或者后续状态等效

#### 2）不完全给定同步时序电路状态表的化简

**状态相容**定义：
设S1 和 S2 是不完全给定时序电路M1和M2 (M1和 M2可以是同一个电路)的两个状态，以其作为初态同时加入**预定的允许输入序列**(除最后一个次态外，其他次态都是确定的)，所产生的输出序列一致，认为确定的输出与对应的不确定的输出相同，则状态 S1 和 S2 是相容的。

我们称 S1和 S2 是相容对。  记为： (S1，S2)。

状态相容无传递性。

![第3章 时序逻辑电路__28](https://pub-e7c93c0b727d457c895b016a26707158.r2.dev/2026/05/%E7%AC%AC3%E7%AB%A0%20%E6%97%B6%E5%BA%8F%E9%80%BB%E8%BE%91%E7%94%B5%E8%B7%AF__28.png)

任意两边需要满足相连，那么可以称为相容类。

##### 最小化相容状态集确定

1. 覆盖性：包含全部的原始状态
2. 闭合性：任意相容类的次态应仅属于该集中的一个相容类
3. 最小化条件：数目最少（经验性规则）

**方法**：
1. **隐含表**->相容对
2. **合并图**->最大相容类
3. **覆盖闭合图**->最小化

### 3 状态分配

为最小化状态表中的每个字母状态赋予一组二进制代码，称为**状态编码**。
#### 1）状态编码的基本问题

所需触发器位数：

$$K = \lceil \log_2 n \rceil$$

其中 $n$ 为最小化后的状态总数，$K$ 为所需触发器个数，$\lceil \cdot \rceil$ 表示向上取整。$K$ 个触发器共有 $2^K$ 种编码，其中 $2^K - n$ 个为无效状态（未使用状态）。

![第3章 时序逻辑电路__31](https://pub-e7c93c0b727d457c895b016a26707158.r2.dev/2026/05/%E7%AC%AC3%E7%AB%A0%20%E6%97%B6%E5%BA%8F%E9%80%BB%E8%BE%91%E7%94%B5%E8%B7%AF__31.png)

不同编码方案的区别在于状态之间的**相邻关系**不同，这会直接影响激励函数和输出函数的卡诺图复杂度，进而决定电路的逻辑门级实现成本。

**相邻**：两个二进制代码仅有**一位不同**，称为相邻。例如 00 与 01、10 相邻，但与 11 不相邻。

#### 2）相邻状态分配法

**核心思想**：通过合理安排状态的二进制编码，使次态函数和输出函数的卡诺图上逻辑相邻的项在几何位置上相邻，从而简化表达式。

相邻判定规则（三规则法）：

---

**规则 I**：在**相同输入条件**下，若两个现态的次态相同，则这两个现态应分配相邻代码。
改善效果：$K \times R$
  - $K$：触发器位数（状态变量个数）
  - $R$：满足该规则的**现态对**对数


---

**规则 II**：对于**同一个现态**，若在**相邻输入条件**下（即输入仅有一位不同）的次态也相邻，则安排这两个次态相邻编码。

改善效果：$(K - 1) \times m$
  - $m$：满足该规则的**相邻输入对**数
  - $K - 1$：因为次态相邻关系涉及次态编码的 $K$ 位，但其中一位可能由输入直接决定，剩余 $K-1$ 位受编码影响


---

**规则 III**：若两个现态的**输出完全相同**，则安排这两个现态相邻编码。


改善效果：$(p \times q) \times l$
  - $p$：输入组合数
  - $q$：输出变量个数
  - $l$：满足该规则的**现态对**对数


---

**总改善效果**：$$E_{total} = E_I + E_{II} + E_{III}$$

选择使总改善效果最大的编码方案。

## 脉冲异步时序电路的设计

设计步骤基本上和同步时序电路的设计步骤一样，但是要采用简化的状态表和状态图。最后要检查是否存在**挂起**现象。

>[!TIP]
>**时钟信号**CLK是现态y和输入x的函数，而各触发器的**输入控制信号**Y应仅为现态y的函数。

状态不变时，令`CLK=0`，有利于激励函数的化简。

![第3章 时序逻辑电路__37](https://pub-e7c93c0b727d457c895b016a26707158.r2.dev/2026/06/%E7%AC%AC3%E7%AB%A0%20%E6%97%B6%E5%BA%8F%E9%80%BB%E8%BE%91%E7%94%B5%E8%B7%AF__37.png)

这张图展示的是在使用门控时钟的 D 触发器设计时序逻辑电路时，如何从状态表推导出控制函数D输入、CLK和输出函数（Z）的卡诺图。
### 1. 明确坐标映射与扩充状态表

首先需要将原始状态表映射到卡诺图的二维坐标系中：

**行坐标（现态 $y_1y_2$）：** 状态表中只有 00、01、11 三个状态。缺失的 10 是未使用的无效状态，因此在所有的卡诺图中，$y_1y_2=10$ 这一行全部填入无关项 `d`。

**列坐标（输入 $X_1X_2$）：** 状态表上方用黄色高亮标出了 10 和 01。这表明表中的 $x_1$ 列对应输入 $X_1X_2=10$，$x_2$ 列对应输入 $X_1X_2=01$。对于未在表中出现的输入 $X_1X_2=00$ 和 $X_1X_2=11$，在卡诺图对应的列中，将其视为无关项 `d` 。
### 2. 确定时钟函数 

为了降低功耗或避免竞争冒险，只有当触发器的状态需要发生翻转时，才给它一个时钟脉冲。

-  比较现态 $y_i^n$ 和次态 $y_i^{n+1}$。
    
    如果 $y_i^n \neq y_i^{n+1}$（状态需要改变），则必须有触发边沿，此时 $CLK_i = 1$。
    
    如果 $y_i^n = y_i^{n+1}$（状态保持不变），则不需要触发，此时 $CLK_i = 0$（触发器靠自身特性保持原值）。
    
- 以 $CLK_1$ 为例：
    
    现态 $y_1y_2=01$，输入 $x_1(10)$ 时，表中次态为 $11$。$y_1$ 需要从 0 变到 1，必须触发，所以 $CLK_1$ 卡诺图的 `(01, 10)` 位置填 1。
    
    现态 $y_1y_2=00$，输入 $x_1(10)$ 时，表中次态为 $01$。$y_1$ 从 0 保持为 0，无需改变，所以 $CLK_1$ 卡诺图的 `(00, 10)` 位置填 0。
    

### 3. 确定控制函数 ($D_1$, $D_2$)

这是该设计思路中最精妙的一步，它利用了门控时钟的特性来极大化简 D 端的逻辑。

- **判断规则：**
    
    当 $CLK_i = 1$ 时（时钟有效） $D_i = y_i^{n+1}$。
    
    当 $CLK_i = 0$ 时（时钟被屏蔽）应位置的 D 值**全部视为无关项 d**。
    
- 以 $D_1$ 为例：
    
    在刚才求出的 (01, 10)位置，$CLK_1 = 1$，且期望的次态 $y_1^{n+1}$ 是 1，所以 $D_1$ 卡诺图在该位置填 1。
    
    在 (00, 10) 位置，$CLK_1 = 0$。因为时钟无效，输入什么都不影响，所以 $D_1$ 卡诺图在该位置直接填 d。

最后要检查是否存在**挂起**，如果存在，则需要对d进行合理赋值。
