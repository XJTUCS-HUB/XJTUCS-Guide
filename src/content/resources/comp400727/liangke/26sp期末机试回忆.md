## 选择题(10*2=20)

数制转换。

异常控制流。根据fork函数画图表，然后判断输出是否存在。
类似于：
```c
int main(int argc,char** argv){
    printf("begin");
    pid_t pid;
    int x=1;
    pid=Fork();
    if(pid==0){
        printf("child:x=%d\n",++x);
        return 0;
    }
    printf("parent:x=%d\n",--x);
    return 0;
}
```

时间局部性和空间局部性。给两段代码，让你比较谁的时间局部性、空间局部性更好。
