# 隐私说明

本站只收发布课程资料和经验所需的信息。作者可以使用真名、网名或匿名，联系方式默认不公开。

请勿提交学号、身份证、成绩单、手机号、私人邮箱、私人账号、精确住址、Token、密钥、群聊截图或未经同意的他人信息。

GitHub Issue 和 Pull Request 是公开记录。提交前请删除正文、截图、文件名和图片元数据中的个人信息。

发现隐私问题时不要在公开 Issue 中复述敏感内容，请发送邮件至 [XJTUCS-HUB@protonmail.com](mailto:XJTUCS-HUB@protonmail.com)，并按 [TAKEDOWN.md](TAKEDOWN.md) 的流程处理。
