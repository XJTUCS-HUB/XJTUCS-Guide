# XJTUCS Guide 大致框架

## 1. 项目定位

XJTUCS Guide 是由西安交通大学计算机专业学生维护的非官方资料站，主要整理课程信息、学习资料和修课经验，同时保留少量推免经验帖。

## 3. 课程页面

每门课使用一个固定页面，主要包含：

1. 课程名称、代码、学分和分类；
2. 课程简介与学习目标；
3. 先修建议和相关课程；
4. 笔记、实验、考核与复习资料；
5. 带课程背景的学习体验；
6. 外部资源、纠错和贡献入口。

学习体验只写具体情况和个人感受，不设置评分、排名或聚合评价。

## 4. 推免经验

推免部分参考 Xjtuse-Guide 的独立经验帖形式，按届次建立简单目录。

每篇文章可以包含准备过程、申请经历、踩坑和个人建议。作者可以使用真名、网名或匿名。

网站不维护推免名单、联系方式、成绩排名、去向统计、政策数据库和 QQ 群。

## 5. 内容组织

首版只使用三个内容集合：

| 集合 | 内容 |
| --- | --- |
| `courses` | 课程代码、名称、分类、学分和简介 |
| `resources` | 笔记、实验、考核、复习和学习体验 |
| `recommendation-experiences` | 推免经验帖 |

所有内容使用 Markdown 或 MDX 保存。作者、来源和更新时间是基本字段，学期、教师、届次等信息按需填写。

## 6. 贡献方式

- 不熟悉 Git：通过 GitHub Issue Form 投稿；
- 会简单 Markdown：使用“编辑此页”；
- 熟悉 Git：直接提交 Pull Request；
- 隐私、版权和下架问题：通过公共邮箱 XJTUCS-HUB@protonmail.com 联系。

首版不开发复杂投稿后台，也不接受来源不明的大文件和附件。

## 7. 技术方案

| 部分 | 方案 |
| --- | --- |
| 框架 | Astro + TypeScript |
| 内容 | Markdown/MDX + Content Collections |
| 导航 | 课程目录优先的静态入口 |
| 样式 | Design Tokens + Astro 组件局部样式 |
| 数学公式 | KaTeX |
| 图表 | Mermaid |
| 部署 | GitHub Actions + GitHub Pages |

整个网站保持静态，不使用数据库和自有后端。
