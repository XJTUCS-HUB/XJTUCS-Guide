# 投诉与下架

事实错误、错字和普通失效链接可以使用公开的[纠错表单](https://github.com/XJTUCS-HUB/XJTUCS-Guide/issues/new?template=correction.yml)。

隐私、版权、名誉、安全和当前考试泄题问题不要发公开 Issue。请发送邮件至 [XJTUCS-HUB@protonmail.com](mailto:XJTUCS-HUB@protonmail.com)，不要在公开渠道复述敏感内容。

处理流程：

```text
收到报告 -> 判断是否先隐藏 -> 核对内容和来源 -> 修改、恢复或删除 -> 记录处理结果
```

涉及个人隐私、当前考试内容、密钥或明显侵权材料时，维护者可以先隐藏内容，再完成核实。

报告时请准备：页面 URL、问题类型、权利关系或授权依据，以及希望采取的处理方式。敏感证明只通过确认后的私密渠道发送。
